# 🎲 Random Quote Generator

A simple web app that displays a random inspirational quote each time the user clicks a button. Perfect for learning JavaScript, APIs, and DOM manipulation.

## 🚀 Features

- Generates a random quote instantly
- Displays quote text and author
- Clean and responsive UI
- Button to generate a new quote
- Button to  Copy quote

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript 
- API Ninjas Quote APi
